# left-f-tools

> js 工具函数库



### Install

```shell
npm i --save left-f-tools

or

yarn add left-f-tools
```





### Usage

```js
import { Enum } from 'left-f-tools'

const type = new Enum([
    [ 1, 'wait' ],
    [ 2, 'run' ],
    [ 3, 'stop' ]
])
```
